## Resumo do Fix

O componente `TOConlineRelatorios` estava a tratar a resposta da API como formato JSON:API (com `data[].attributes`), mas a API TOConline retorna objetos simples com campos diretamente acessíveis. Exemplo de resposta real da API:

```json
{
  "document_no": "FT 2026/1",
  "gross_total": 11843,
  "customer_business_name": "CALDERERIA KORTABERRI,S.L",
  "date": "2026-01-15"
}
```
