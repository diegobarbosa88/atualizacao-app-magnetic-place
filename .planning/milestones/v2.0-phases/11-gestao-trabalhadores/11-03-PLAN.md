Criar tabela para histórico de alterações do valor hora do trabalhador:

```
Table: worker_valorhora_history
Columns:
- id: uuid (primary key)
- worker_id: uuid (foreign key to workers)
- valor_anterior: numeric (valor antes da alteração)
- valor_novo: numeric (novo valor)
- data_alteracao: timestamp (quando ocorreu)
- alterado_por: text (opcional - quem fez a alteração)
```

Esta tabela permite consultar a evolução temporal do valor hora de cada trabalhador.

Na TeamContext, adicionar função para criar registos de histórico:
```javascript
const saveValorHoraHistory = async (workerId, valorNovo) => {
  const worker = workers.find(w => w.id === workerId);
  if (!worker || worker.valorHora === valorNovo) return;  // Sem alteração
  
  await saveToDb('worker_valorhora_history', null, {
    worker_id: workerId,
    valor_anterior: worker.valorHora,
    valor_novo: valorNovo,
    data_alteracao: new Date().toISOString()
  });
};
```
