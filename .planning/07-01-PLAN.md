# Plano: Fase 7 - Modularização Admin (Equipa e Clientes)

Este plano foca na extração das funcionalidades de gestão de trabalhadores e clientes para componentes dedicados.

## 🎯 Objetivos
- Criar componentes independentes para a aba "Equipa".
- Criar componentes independentes para a aba "Clientes".
- Limpar a lógica de formulários (workerForm/clientForm) do `app.jsx`.

## 🛠️ Tarefas

### 1. Preparar Diretórios
- [ ] Criar diretório `src/features/admin`.

### 2. Extrair Gestão de Equipa
- [ ] Criar `src/features/admin/TeamManager.jsx`.
- [ ] Mover a lógica de `workerForm` e a renderização da tabela de trabalhadores.
- [ ] Consumir `workers`, `schedules`, `clients` e `saveToDb`/`handleDelete` via `useApp()`.

### 3. Extrair Gestão de Clientes
- [ ] Criar `src/features/admin/ClientManager.jsx`.
- [ ] Mover a lógica de `clientForm` e a renderização da tabela de clientes.
- [ ] Consumir `clients`, `saveToDb`/`handleDelete` via `useApp()`.

### 4. Integração no AdminDashboard
- [ ] Substituir o código inlined no `AdminDashboard` pelos novos componentes.
- [ ] Validar a navegação entre abas.

## 🧪 Verificação
