## Fluxo Completo (para referência)

```
Trabalhador abre portal
  → effect detecta: sem log hoje + defaultClientId existe
  → card "Registar Entrada" aparece
  → clica → GPS capturado (check_in_lat/lng) → log gravado com startTime

Card "Em serviço" aparece
  → timer ao vivo (atualiza cada 30s)
  → botão "Iniciar Pausa" → GPS capturado (break_start_lat/lng)
  → card muda para "Em pausa Xmin" + botão "Terminar Pausa"
  → "Terminar Pausa" → GPS (break_end_lat/lng)
  → botão "Registar Saída" → GPS (check_out_lat/lng) → endTime gravado

Portal do cliente → tab "Hoje"
  → fetch directo Supabase (independente do mês seleccionado)
  → card por trabalhador: 4 blocos (Entrada / Pausa / Saída / Total)
  → cada bloco com "📍 Ver mapa" se coordenadas disponíveis
  → actualiza via real-time Supabase (lastRealtimeUpdate)
```
